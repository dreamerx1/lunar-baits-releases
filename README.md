# lunar-baits-releases

Public release artifacts for **LunarBaits** — Lunar Baits gamybos
valdymo sistema. The application source is private at
[dreamerx1/lunar-baits](https://github.com/dreamerx1/lunar-baits); this
repo only carries the built Windows exe and a `version.json` manifest
that the running app polls for updates.

Each release tag (`vX.Y.Z`) attaches `LunarBaits.exe`. The
`version.json` on the `main` branch always points at the latest one.
